BUSY Bar est un dispositif multi-outil de productivité avec un écran LED pixelisé. Minuteur de concentration avec fonction de blocage des distractions sur votre téléphone et PC. Entiérement personnalisable, open-source et pret pour la maison connectée.

Outil de productivité

Blocage des disttractions, message BUSY personnalisé, minuteur de concentration, synchronisation multiplateforme

Applications et intégration

Bibliothèque d'applications, connexion à des logiciels tiers, intégrations avec les événements de calendrier et les appels

Support maison connectée

Connectez-vous à Google Home et Apple Home via e protocole Matter

Pour les développeurs

API HTTP ouverte, SDK open-source, bibliothèques Python / Go / Javascript, MQTT, sans verrouillage fournisseur

Montrer que vous etes occupé

Empechez les autres d'interrompre votre concentration profonde. L'écran affiche que vous etes occupé et quand vous serez à nouveau disponible.

Activation instantanée

Activez et désacativez le statut BUSY manuellement avec un large bouton. Fonctionne entiérement hors ligne, sans logiciel supplémentaire requis.

Set timer

Set an optinal expiration timer BUSY using the scroll wheel to let others know when you'll be available again.
